Crosspatch designer

This tool allows you to create and print crosspatch patterns.
Building this design tool will enable you to test some interesting parameters and learn a lot for your first 3d food printing tool!
This app will generate a file that with the data needed to 3D print a food crosshatch pattern.
The control of the app is done with the following input parameters:
Number of rows
Number of columns
Number of layers
Crosspathc width
Crosspathc length
This five parameter will control the dimension and shape of the crosshatch

  • Layer heigth

  • Z travel height

  • Material diameter

  • Noozle diameter

  • Feedrate:

    • Travel

    • Printing

    • Extruder

  • Extruder:

    • Retraction

    • Feedrate

    • Build up pressure

    • Release pressure
